upload(\ElKuKu\Crowdin\Languagefile  $languagefile, string  $language, boolean  $importDuplicates = false, boolean  $importEqualSuggestions = false, boolean  $autoImproveImports = false) : \Psr\Http\Message\ResponseInterface

Upload existing translations to your Crowdin project.

Parameters

\ElKuKu\Crowdin\Languagefile $languagefile

The translation object.
string $language

The language tag.
boolean $importDuplicates

Defines whether to add translation if there is the same translation previously added. Acceptable values are: 0 or 1. Default is 0.
boolean $importEqualSuggestions

Defines whether to add translation if it is equal to source string at Crowdin. Acceptable values are: 0 or 1. Default is 0.
boolean $autoImproveImports

Mark uploaded translations as approved. Acceptable values are: 0 or 1. Default is 0.

export()

export(string  $branch = '') : \Psr\Http\Message\ResponseInterface

Build ZIP archive with the latest translations. Please note that this method can be invoked only once per 30 minutes (there is no such restriction for organization plans). Also API call will be ignored if there were no changes in the project since previous export. You can see whether ZIP archive with latest translations was actually build by status attribute ("built" or "skipped") returned in response.

Parameters

string $branch

The name of related version branch.

Returns

\Psr\Http\Message\ResponseInterface

See also
https://crowdin.com/page/api/export

download()

download(string  $package, string  $toPath, string  $branch = '') : \Psr\Http\Message\ResponseInterface

Download ZIP file with translations. You can choose the language of translation you need or download all of them at once.

Note: If you would like to download the most recent translations you may want to use export API method before downloading.

Parameters

string $package

Language code or "all" to download a bundle with translations to all languages.
string $toPath

Local path where to download the translation package.
string $branch

The name of related version branch.

\Psr\Http\Message\ResponseInterface
